## Comandos Comuns

Abaixo o comando para criar um enviroment python

```zsh
python3 -m venv <nome-do-ambiente>
```

```zsh
 source <caminho-onde-foi-criado-enviroment-python>/<nome-do-ambiente>/bin/activate
```
